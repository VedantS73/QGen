# **QGen**
#### QGen - Question Paper Generator
<br>

## Requirements :
#### Python 3.10+
#### Django 4.2.1+
#### Reportlab
<br>

## How to Run :
#### Open Terminal in project root directory.
#### Type the following : `python manage.py runserver`
<br>

## Default Credentials :
#### username : admin
#### password : admin
